# Esp32S3-Radar-Project
I made Esp32s3 radar project. Already display scanned are on ILI9341 tft Screen.

[![V1](https://img.youtube.com/vi/t4QVxeeEtEQ/0.jpg)](https://youtu.be/t4QVxeeEtEQ)

## Pin Configuration

 TRIG_PIN      7
 
 ECHO_PIN      16 
 
 SERVO_PIN     17 

 TFT_MISO      13
 
 TFT_MOSI      11
 
 TFT_SCLK      12
 
 TFT_CS        4
 
 TFT_DC        15
 
 TFT_RST       9
 
## Using Parts

Esp32S3 Development Board

SG90 Micro Servo Motor

ILI9341 Display Module

HC-SR04 Ultrasonic Distance Snsor

Half Sized BreadBoard
